# 농사 게임

마인크래프트의 농사를 2d로 만든게임입니다.

## 설치

```console
$ git clone https://github.com/newkincode/simple_farming_game.git
$ cd simple_farming_game
$ (pip) install -r Pipfile

$ python main.py
```

## 빌드

pyinstaller라이브러리가 필요합니다.

```console
$ pyinstaller main.py
```

## 조작법

-   방향키: 이동
-   `E`: 낫. 흙을 농작지로 바꿉니다.
-   `S`: 삽. 농작지를 흙으로 되돌립니다.
-   `F`: 괭이. 수확에 사용합니다.
-   `R`: 씨앗. 농작지에 심습니다.
-   `SPACE`: 달리기
-   `Z`: 선택 해제
-   `D`: 선택 항목 사용
