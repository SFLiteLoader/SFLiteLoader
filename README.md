# 농사 게임

마인크래프트의 농사를 2d로 만든게임입니다.

## 도와주신분

ky0422 - 오류코드 1 해결

## 설치

```console
$ git clone https://github.com/newkincode/simple_farming_game.git
$ cd simple_farming_game
$ (pip) install -r requirements.txt

$ python main.py
```

## 조작법

-   방향키: 이동
-   `Z`: 아무것도 선택안함
-   `F`: 괭이 선택
-   `R`: 씨앗 선택
-   `Z`: 선택 해제
-   `D`: 선택 항목 사용
