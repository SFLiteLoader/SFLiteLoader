# 농사 게임

모든걸 농사짓자!

## 설치

```console
$ git clone https://github.com/newkincode/simple_farming_game.git
$ cd simple_farming_game
$ pip install pygame

$ python main.py
```

## 빌드

```console
> ./build.bat
```
이후 lib폴더와 assets 폴더와 data폴더를 build폴더 안 main폴더에 넣어줍니다(복사 붙여넣기)

## 조작법

-   방향키: 이동
-   `E`: 낫. 흙을 농작지로 바꿉니다.
-   `S`: 삽. 농작지를 흙으로 되돌립니다.
-   `F`: 괭이. 수확에 사용합니다.
-   `R`: 씨앗. 농작지에 심습니다.
-   `SPACE`: 달리기
-   `Z`: 선택 해제
-   `D`: 선택 항목 사용
-   `G`: 아이템 정보
-   `W`: 물뿌리게
-   `V`: 비타민
-   `A`: 판매
-   `B`: 구매
-   `H`: 도움말
-   `K`: 저장
-   `L`: 불러오기
