# 농사 게임

모든걸 농사짓자!

## 설치

```console
$ git clone https://github.com/newkincode/simple_farming_game.git
$ cd simple_farming_game
$ pip install pygame

$ python main.py
```

## 빌드

```console
> ./build.bat
```
이후 lib폴더와 assets 폴더와 data폴더를 build폴더 안 main폴더에 넣어줍니다(복사 붙여넣기)
